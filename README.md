# Kanban Task Board

A Trello-style task management board built using React + Vite.

## Features
- Add Tasks
- Delete Tasks
- Move Tasks Between Columns
- Inline Editing
- Priority System (High / Medium / Low)
- LocalStorage Persistence

## Tech Stack
- React
- Vite
- JavaScript
- CSS
- LocalStorage
